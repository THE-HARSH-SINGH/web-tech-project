Portfolio Project

Overview
This project is a multi-page student portfolio that demonstrates HTML, CSS, JavaScript, and PHP mini applications.

Features
- Portfolio landing page with profile, summary, and highlights
- HTML-only internship application form
- CSS-driven to-do UI with JavaScript interactivity
- JavaScript calculator with history and keyboard input
- PHP login and form handling demo

Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- PHP
- Apache (XAMPP)

Setup
1. Place this folder inside your XAMPP htdocs directory.
2. Start Apache from the XAMPP control panel.
3. Open the project in a browser:
   http://localhost/Portfolio_Project/

If Apache is running on a different port (for example 8080):
http://localhost:8080/Portfolio_Project/

Pages
- Home: http://localhost/Portfolio_Project/index.html
- Page 2 (Internship form): http://localhost/Portfolio_Project/page2.html
- Page 3 (CSS demo): http://localhost/Portfolio_Project/page3.html
- Page 4 (Calculator): http://localhost/Portfolio_Project/page4.html
- Page 5 (PHP): http://localhost/Portfolio_Project/page5.php

Assets
Place the profile image at:
images/profile.png
